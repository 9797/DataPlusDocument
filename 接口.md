### 页面组
1. 增加页面组

| 参数        | 含义           | 类型  |
| ------------- |:-------------:| -----:|
| name      | 项目组名称 | String |

2. 删除项目组

| 参数        | 含义           | 类型  |
| ------------- |:-------------:| -----:|
| id      | 项目组id | Number |

3. 获取项目组信息

| 参数        | 含义           | 类型  | 必要性  |
| ------------- |:-------------:| -----:| -----:|
| id      | 项目组id | Number | 必要|

4. 修改项目组信息

| 参数        | 含义           | 类型  | 必要性  |
| ------------- |:-------------:| -----:| -----:|
| id      | 项目组id | Number | 必要|
| name      | 项目组名称 | String | 非必要|
| data      | 项目组数据 | Object | 非必要|
